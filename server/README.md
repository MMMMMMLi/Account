# Account-Server

| SpringCloud | Hoxton.SR8 | 直接使用IDEA构建的项目，使用的是最新版。 |
| MySQL | 5.6.40 | 之前搭建的一套Mysql环境，没有更改。 |

| Druid | 1.1.23 | 数据池管理工具。 |
